# Containerized-WordPress-and-MySQL-Deployment
# WordPress and MySQL Multi-Container Deployment using Docker Compose

## Project Overview

This project demonstrates the deployment of a multi-container WordPress application using Docker Compose. The application consists of a WordPress container and a MySQL database container running on AWS EC2.

## Technologies Used

* Docker
* Docker Compose
* WordPress
* MySQL
* AWS EC2
* Ubuntu Linux

## Architecture

WordPress Container ↔ MySQL Container

## Features

* Multi-container deployment
* Container networking
* Database integration
* Docker Compose orchestration
* AWS EC2 hosting

## Deployment Steps

1. Install Docker and Docker Compose.
2. Create the docker-compose.yaml file.
3. Run:

docker compose up -d

4. Access WordPress through the EC2 public IP and configured port.

## Outcome

Successfully deployed and managed a containerized WordPress application using Docker Compose, demonstrating containerization and infrastructure deployment skills.
